# Python + Qt5 Calculator

A small calculator developed in Python with a Qt5 graphical interface.

## Features

- Addition, subtraction, multiplication, division
- Simple graphical interface

## How to use

1. Install Python 3
2. Install PyQt5: `pip install PyQt5`
3. Run `calculatrice.py`
